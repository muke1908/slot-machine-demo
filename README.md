## Slot Machine

This is a slot machine implementation having following features:
- Configurable reels (./config/slot, ./config/reel)
- Supports bias in each reel
- Configurable pay tables (./config/paytable)
- Balance indicator
- Debug mode



## Installation  
```npm start```
