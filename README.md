## SLOT Machine

This is a slot machine implementation having following features:  
[x] - Configurable reels (./config/slot, ./config/reel)
[x] - Supports bias in each reel
[x] - Configurable pay tables (./config/paytable)
[x] - Balance indicator
[x] - Debug mode



## Installation  
```npm start```
